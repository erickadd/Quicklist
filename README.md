# 🛒 Quicklist - Gerenciador de Lista de Compras

![Status](https://img.shields.io/badge/status-concluído-brightgreen)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

Aplicação simples e intuitiva para gerenciar sua lista de compras da semana. Adicione, marque como concluído e remova itens com facilidade.

## ✨ Funcionalidades

- ✅ **Adicionar itens** - Campo de input com botão ou tecla Enter
- ✅ **Marcar como concluído** - Checkbox com feedback visual (texto riscado)
- ✅ **Remover itens** - Botão com ícone de lixeira e animação de remoção
- ✅ **Itens pré-cadastrados** - Lista inicial com exemplos
- ✅ **Contador de itens** - Mostra total e quantos concluídos
- ✅ **Mensagens de feedback** - Alertas para ações realizadas
- ✅ **Validação** - Não permite itens vazios
- ✅ **Responsividade** - Funciona em desktop e mobile
- ✅ **Animações CSS** - Transições suaves e feedback visual
